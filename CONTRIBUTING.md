# Contributing

Issues and pull requests are welcome.

## Code of Conduct

Be respectful. We follow the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## How to contribute

1. Fork the repository.
2. Create a branch: `git checkout -b feat/your-feature`.
3. Make your changes and ensure tests pass: `pnpm test`.
4. Open a pull request against `main`.

## Reporting issues

Use GitHub Issues. For security vulnerabilities, email ops@vauban.tech directly.
