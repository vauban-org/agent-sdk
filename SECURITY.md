# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do NOT open a public GitHub issue.**

Send a report to: security@vauban.tech

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 48 hours and coordinate a fix. Once resolved, we will credit you in the release notes if desired.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.17.x (beta) | Yes |
| < 0.16 | No |
